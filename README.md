<h1 align="center">Pets Api</h1>

## :computer: Projeto

- [C#](https://docs.microsoft.com/pt-br/dotnet/csharp/getting-started/)
- [Entity Framework Core v3.1.9](https://docs.microsoft.com/pt-br/ef/core/)
- [Swagger v5.6.3](https://swagger.io/)
